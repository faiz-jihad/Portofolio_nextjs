# 🌌 Cinematic Developer Portfolio

A high-end, immersive "Scrollytelling" developer portfolio built with **Next.js 15**, **Framer Motion**, and **HTML5 Canvas**. This project bridges the gap between creative design and robust engineering, featuring real-time GitHub integration and futuristic animations.

---

## ✨ Key Features

- 🎞️ **Cinematic Scrollytelling**: An immersive experience using HTML5 Canvas to scrub through high-quality image sequences based on scroll position.
- 🚀 **Futuristic Preloader**: A custom-built system initialization sequence with animated status updates and progress tracking.
- 📊 **Real-time GitHub Stats**: Comprehensive data fetching via GitHub REST & GraphQL APIs, including total stars, forks, and a visual language distribution bar.
- 🎡 **Infinite Marquee Carousel**: An automated, dual-row repository showcase that seamlessly loops all open-source projects.
- 📌 **Pinned Projects**: A grid of featured repositories with elegant glassmorphism effects and hover glow.
- 📈 **Dynamic Activity Panel**: Visual representation of GitHub contributions and a recent activity timeline.
- 📱 **Fully Responsive**: Optimized for all devices, from mobile phones to ultra-wide displays.
- 📬 **Interactive Contact Form**: Integrated with Supabase for reliable message delivery and state management.

---

## 🛠️ Tech Stack

### Frontend & UI
- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Canvas Rendering**: Native HTML5 Canvas for frame sequence playback
- **Iconography**: [Iconify](https://iconify.design/) for uniform, high-performance icons

### Backend & Integration
- **Database**: [Supabase](https://supabase.com/)
- **Data Source**: GitHub REST API & GraphQL API
- **Deployment**: Vercel (Optimized for Next.js)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- GitHub Personal Access Token (PAT)
- Supabase Project (URL & Anon Key)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/faiz-jihad/Portofolio_nextjs.git
   cd portoweb
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env.local` file:
   ```env
   GITHUB_TOKEN=your_github_pat_here
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run development server**
   ```bash
   npm run dev
   ```
